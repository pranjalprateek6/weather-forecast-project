Features.

A weather forecast web app that uses the OpenWeatherMap API allow users to search for current weather conditions by city or location. The app would make a call to the OpenWeatherMap API, providing the app with current weather data such as temperature, precipitation, wind speed, and more. This data would then be displayed to the user in an easy-to-read format, which could include a detailed report and a seven-day forecast.

Left Container

The left panel consists :
• Day and date, current exact temperature as well as next 3 hours of temperature in half hour gaps.
• Pressure.
• Wind speed.
• Humidity.
• Sunrise and Sunset time.

Right Container

The right panel consists :
• The temperature details of the whole next week with respect to the current date.
• The highest and the lowest temperature of the day.
• Displays the information about the weather description (Scattered clouds, Light rain, Clear sky).
