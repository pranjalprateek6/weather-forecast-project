About Us

This web app is made by Pranjal Prateek and Paawan Gaba of BCA in 5th semester for Minor Project. Our project is a weather forecast web app that uses the OpenWeatherMap API allow users to search for current weather conditions by city or location. The technologies used to create the app are the OpenWeatherMap API, HTML, CSS, JavaScript and php.
